Comparing Classification Models¶

On this part, we trained the various classification models with the Million Song Datataset data and compare its accuracy. We then use the model with the highest accuracy to predict the "hotness" of a music using Spotify API. We compared the following models and identified the one with the highest accuracy.

  • Random forest classifier
  • XGBoost classifier (XGB)
  • Logistic regression
  • k-Nearest neighbor (kNN)
  • Support vector machine (SVC)
  • Decision tree
  • Stochastic gradient descent (SGD)

We are dependent on the songs that have appeared on the Billboard Hot 100 chart as our training data and we will use this to make predictions of Spotify music's hotness.

# Function to flag rows of songs that appear in the hot 100 data (1)
# and 0 if otherwise.
def find_target(row):
    lst = bb_tracks
    if row['title'] in lst:
        return 1
    else:
        return 0

def testingModel(model, X_train, Y_train):
    scores = cross_val_score(model, X_train, Y_train, cv=10, scoring = "roc_auc")

    print("Mean:", scores.mean())
    print("Standard Deviation:", scores.std())
    return scores.mean()
