Part 2: Identifying Popular Music with Billboard Hot 100 Data¶

The original data in A Million Songs dataset came with a song hotness feature. However, around 4500 songs were missing this feature, which is almost half of the subset we were using. Thus, we wanted to find a new way to classify if a song is a hit or not. We decided to use BillBoard Top 100 to determine popularity. If a song has appeared on Top 100 BillBoard at least once, then it will be classified as a hit song.

I wrote python scripts using BeautifulSoup to scrape billboard.com and get all the songs that appeared on the chart from 1958 to 2012. I stopped at 2012 since to our most recent songs in the dataset were released in 2012. After getting the list of songs that have been on billboard, we go back to our million songs dataset, and classified them accordingly. Out of million songs in our dataset 186,958 songs were classified as hot songs.

Scrape Billboard Hot 100 data from Billboard website from 1958 (inception) up to 2012 (latest year in the dataset). We then save the scraped data in a CSV file.

# Function to initiate request on each url and return an lxml file
def get_soup(start_url):
    response = requests.get(start_url)
    page = response.text
    return BeautifulSoup(page, "lxml")
In [ ]:
# Library imports
from datetime import date, datetime, timedelta

# A function to create list of Billboard Hot 100 urls that catalogue each song going back to 1958
def perdelta(start, end, delta):
    curr = start
    while curr < end:
        yield curr
        curr += delta

# Create an empty list. We will be appending urls here.
urls = []

# Generate chart urls from 9 Aug 1958 to 31 Dec 2012
for suffix in perdelta(date(1958, 8, 9), date(2012, 12, 31), timedelta(days=7)):
    prefix = 'http://www.billboard.com/charts/hot-100/'
    link = prefix + str(suffix)
    urls.append(link)

# Function to scrape artists from the soup data
def get_artist(soup):
    artist = soup.find_all(class_="chart-row__artist")
    artists = []
    for item in artist:
        item = item.text.strip()
        artists.append(item)
    return artists

# Function to scrape track name from the soup data
def get_track(soup):
    track = soup.find_all(class_="chart-row__song")
    tracks = []
    for item in track:
        item = item.text.strip()
        tracks.append(item)
    return tracks

    # Get the latitude dataframe cell and store in a variable
    latitude = artist_latitude.iloc[i]
    # Get the longitude dataframe cell and store in a variable
    longitude = artist_longitude.iloc[i]
    
    # There are artist_longitude and artist_latitude that are 
    # null so if we detect at least one null of both values

    if (math.isnan(latitude) or math.isnan(longitude)):
        countries.append('')
    # If both artist_longitude and artist_latitude values 
    # are present, we do a reverse geocoding and append  
    # these values into a dataframe
    else:
        coordinates = (latitude, longitude),
        # Do a reverse geocoding

        # Append to list
